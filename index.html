🧠 The Learning Path Explained

Why each topic matters and how deep to go.

Programming & Systems

C++ (The Cherno) — Go very deep. Isaac Sim/Lab are C++. You'll write robot controllers and ROS2 nodes in C++. Master pointers, references, RAII, templates, STL, memory management.
Python — Intermediate level. NumPy + PyTorch. You'll write RL training scripts and rapid prototypes. Not as deep as C++.
Algorithms & DS (NeetCode) — Daily practice for 6+ months. You need LeetCode Medium fluency. Focus on arrays, trees, graphs, DP. These show up in every technical interview.
Computer Systems (CSAPP) — Read chapters 1, 2, 3, 6. Understand memory hierarchy, cache, virtual memory, machine-level code. This is "system architecture" — how the hardware actually works.
Databases — Light coverage. SQL basics (SQLite). Time-series databases for robotics sensor logging. You won't be a DBA, but you need to log robot data.
Memory Management — Deep. Part of The Cherno C++. Stack vs heap, pointers, RAII, smart pointers, memory pools. Real-time robotics requires zero allocations in control loops.

Robotics

ROS2 — Deep. The communication backbone for all Isaac ROS packages. Master nodes, topics, services, actions, TF2, URDF, launch files, ros2_control.
Control Theory — Working knowledge. PID for joint control (Steve Brunton). MPC for trajectory optimization. You don't need to derive proofs, but you need to implement and tune controllers.
Kinematics — Modern Robotics Course 1. Forward/inverse kinematics. Rotation matrices, SO(3), twists, wrenches. The math of robot motion.
Simulators — Deep. Isaac Sim (primary), MuJoCo (secondary). Understand physics engines, contact models, sensor simulation.

AI/ML

PyTorch — Intermediate. Official tutorials + learnpytorch.io. You'll write neural networks, training loops, custom datasets.
Reinforcement Learning — Deep. PPO specifically. Implement from scratch (Spinning Up). This is how you train robot locomotion policies.
Computer Vision — Working knowledge. CS231n lectures. CNNs, object detection. Needed for robot perception.
CUDA — Intermediate. Write your own kernels. Understand memory hierarchy, coalescing, shared memory. Isaac is GPU-accelerated; you need to understand why.
Computer Graphics — Working knowledge. LearnOpenGL + Ray Tracing in One Weekend. Understand the rendering pipeline. Helps you understand Isaac Sim internals.

NVIDIA Stack

Isaac Sim — Deep. The core simulator. All tutorials + build your own scenes.
Isaac Lab — Deep. The RL framework on top of Isaac Sim. Train all your policies here.
Isaac ROS — Deep. Production robot deployment packages. Run perception and navigation pipelines.
Jetson — Working knowledge. Edge deployment target. TensorRT optimization. You won't build custom hardware, but you need to deploy on it.
Sim2Real — Deep. THE key skill for this role. Domain randomization, system identification, policy deployment. Your capstone demonstrates this.
